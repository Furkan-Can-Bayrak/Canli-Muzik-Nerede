# Mekân kapak varsayılan görselleri

Kapak fotoğrafı yüklemeyen işletmeler için kullanılacak görseli **yalnızca buraya** ekleyin.

URL: `{API_BASE_URL}/static/cafes/default.webp`  
Örnek: `http://localhost:3000/static/cafes/default.webp`

(API uç noktası `/cafes` olduğu için görseller `/static/cafes` altında sunulur.)

## Dosya adı

| Dosya          | Kullanım                   |
| -------------- | -------------------------- |
| `default.webp` | Tüm kapaksız mekânlar için |

PNG/JPEG yüklerseniz WebP'ye çevirip `default.webp` olarak kaydedin; kaynak dosyayı silin.

Format: **WebP** önerilir. En az **800×1000 px** (4:5) veya **1200×675 px** (16:9).
